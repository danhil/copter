	Readme File for Code Example:
--------------------------------------------------

PIC32_RTCC\alarm_interrupts

--------------------------------------------------
Code Example Description:
--------------------------------------------------
The purpose of this example code is to demonstrate the PIC32MX peripheral library macros and functions supporting the RTCC module and its various features.

Sets the alarm time and checks that we actually get an alarm. Then sets the alarm time and enables/disables the RTCC alarm interrupts.
