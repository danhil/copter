--------------------------------------------------

PIC32_RTCC\time_date

--------------------------------------------------
Code Example Description:
--------------------------------------------------
The purpose of this example code is to demonstrate the PIC32MX peripheral library macros and functions supporting the RTCC module and its various features.

This example demonstrates multiple ways to initialize the RTCC with the current date. It then shows how to read that date and time with multiple methods. Then it 
configures a function for checking that the RTCC has a clock enabled and counts the time.
